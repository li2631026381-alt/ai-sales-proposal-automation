# AI Sales Proposal Automation

Enterprise-oriented solution to automate structured sales proposals using AI-assisted workflows.

This project provides a simple interface to gather client requirements, estimate ROI, and generate professional proposal documents in PPTX and DOCX formats.

---

## 🚀 Features

✔ Collect client inputs (industry, goals, pain points)  
✔ Calculate proposal preparation savings and ROI  
✔ Generate structured sales proposal content  
✔ Export downloadable DOCX and PPTX files  
✔ Modular, easy to extend solution

---

## 📦 Tech Stack

- **Python 3.10+**
- **Streamlit** — UI interface
- **python-pptx, python-docx** — export document formats
- **pandas** — lightweight data processing
